# `trun`

CLI test utility for Deno and the Browser

## Flags

```sh
--browser -- whether to run tests headlessly in the browser
--concurrency -- number of tests to be run in parallel
--dir -- directory to run tests -- relative to cwd
--filter -- comma delimitated string of files to run -- relative to dir
--ignore -- name of ignore file -- relative to dir
--import-map -- name of deno import map file -- relative to cwd
--output -- name of file to output test results to -- relative to cwd
```
